# SHELLGEI ONLINE JUDGE WEB
シェル芸オンラインジャッジのウェブアプリです。

シェル芸オンラインジャッジ: https://shellgei-online-judge.com/

# HTML生成方法
下記のコマンドでHTMLファイルを生成できます。

```sh
$ cd scripts
$ python3 generate_index_html.py
```
