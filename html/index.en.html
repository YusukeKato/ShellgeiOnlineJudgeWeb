ABOUT SHELL-GEI

About shell-gei

  • Click here
  • Shell-gei is a shell one-liner in the CLI.

Examples of shell-gei

Example 1: Output random String

head -c 65536 /dev/urandom | tr -dc a-zA-Z0-9 | cut -c -16

Example 2: Delete all .jpg files

find . | grep .jpg$ | xargs -I@ rm @

Extra

Extra 1: Python

python3 -c "import math;print(math.pi)"

Extra 2: C++

cat<<EOF>a.cpp
#include <bits/stdc++.h>
int main() {
  printf("C++ Test
");
}
EOF
g++ a.cpp
./a.out
ABOUT RUN OF SHELL-GEI

Environment

  • cat /etc/os-release: Ubuntu 24.04 LTS
  • echo $SHELL: /bin/bash
  • bash --version: 5.2.21(1)-release
  • python3 -V: Python 3.12.3
  • g++ --version: 13.2.0

Attention

  • Input can be get with "cat input.txt".
  • Dangerous shell-gei are prohibited.
  • Extra white space or line breaks will affect correct/incorrect judgment.
  • The output format must follow the expected output.

Limitations

  • Run time : 5.0s
  • Number of characters (in/out) : 1000
